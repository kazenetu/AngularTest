# AngularTest
Angularを体得するためのリポジトリ

# 事前準備

1. Node.jsインストール
1. TypeScriptインストール  
```npm install -g typescript```
1. Angular Cliインストール  
```npm install -g @angular/cli```
1. Angularのプロジェクトを作成する  
   ※プロジェクト名を「practice」とする  
```ng new practice```

# サーバ実行
```ng serve --open```

# メモ
* コンポーネントの作成  
  ```ng g component <コンポーネント名>``` 
* ルーティングの作成  
  ```ng g module app-routing --flat --module=app```
